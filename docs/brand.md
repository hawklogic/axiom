# Axiom Brand Guidelines

## Identity

**Product Name**: Axiom
**Steward**: HawkLogic Systems
**Tagline**: "Deterministic. Inspectable. Offline."

### Naming Rules
- "Axiom" is the product name (standalone)
- "by HawkLogic Systems" is secondary attribution
- Never use "HawkLogic Axiom" — Axiom is the instrument, HawkLogic signs the work

## Visual Identity — ICARUS Aesthetic

### Design Principles
- Flight-control / avionics energy
- Minimal, geometric, sharp lines
- High contrast, precision instruments
- No gradients, no gloss, no rounded playfulness

### Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Deep Slate | #1a1a1a | Primary background |
| Instrument Amber | #f5a623 | Warnings, highlights |
| Signal Cyan | #00d4ff | Links, active states |
| Terminal Green | #00ff88 | Success, valid |
| Error Red | #ff4444 | Errors, critical |
| Neutral Gray | #888888 | Secondary text |
| Pure White | #ffffff | Primary text (dark mode) |

### Typography
- **Monospace** (data): JetBrains Mono, SF Mono, or system monospace
- **Sans-serif** (UI): Inter, SF Pro, or system sans

### Logo Usage
- Primary logo: `assets/brand/axiom-logo.svg`
- Dark variant: `assets/brand/axiom-logo-dark.svg`
- App icon: `assets/brand/axiom-icon.svg`
- HawkLogic badge: `assets/brand/hawklogic-badge.svg` (subtle, secondary)

### Logo Placement
- App icon: Dock, title bar
- Splash screen: Centered, with tagline
- About panel: Large, with version
- README: Header image
